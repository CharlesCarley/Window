◆ x

skMouseAxis skMouse::x

The axis is split into two values absolute and relative. X and Y is the cursor position, and Z is the wheel rotation. The absolute values for the x and y mouse coordinates are relative to the window's left top corner. The absolute z value is a fixed value of +- skWheelDelta. The relative x, y, and z values are between [-1, 1].
