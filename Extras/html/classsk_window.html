skWindow Class Referenceabstract

Detailed Description

skWindow Is the base class for all windows.

Windows do not get instanced directly. Instead, it's the job of the window manager to issue instances based on the desired context.

The context is supplied when the window manager is created.

◆ close()

void skWindow::close ( )

Signals that this window should close.

The window will not close immediately. Instead, it is pushed to a list in the manager, then the next call to process or processInteractive will remove and destroy the window.
